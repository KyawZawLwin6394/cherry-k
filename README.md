# Cherry-K Clinic REST API

## Installation

Use the package manager [NPM] to install dependicies for Cherry-K Clinic API

```nodejs
npm install 
```

## Usage

```nodejs 
# you can either starts the app with 
node server.js

# or you can use nodemon 
nodemon server.js
```
